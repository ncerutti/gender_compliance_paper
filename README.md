# Code for the welfare simulation in: *Do women really comply more? Experimental evidence from a representative sample*

`generate_simulation.ipynb` contains the code for the simulation of the results with different samples.
You will need Python and Jupyter to run it. It relies on the data in `experiment1_long.dta`.

Please refer to the paper for further information.

## Further instructions

The code is made to run with Jupyter notebooks (see https://docs.jupyter.org/en/latest/running.html), which will be installed (if necessary) along the other requirements in `requirements.txt`. Install them with pip install -r requirements.txt. The code has been tested with Python 3.14.5.
